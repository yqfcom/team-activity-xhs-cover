# 团队活动小红书封面 Skill

把团队活动方案、宣传文案、PDF、图片、海报、行程表或聊天记录，转化为有企划感的小红书 / Rednote 3:4 竖版封面提示词。

这个 skill 适合团队活动、小型活动企划、城市探索、主题团建、企业文化活动、非遗体验、旅行路线、打卡任务、社群活动等场景。它的目标不是生成普通企业海报，而是把活动包装成一个有主题、有审美、有参与欲的封面故事。

## 能做什么

- 从内容母版中提炼活动一句话、适合人群、情绪钩子、主题包装和可视化元素。
- 通过逐步选择题确认封面风格、信息结构、素材使用方式和封面文案。
- 支持把风格与信息结构交给模型统一判断。
- 在出图前先输出设计构思，并要求确认视觉元素。
- 生成可直接用于图片生成模型的 3:4 小红书封面提示词。

## 适合输入

- 活动方案文字
- 宣传文案
- PDF 或海报截图
- 行程表、报名页、聊天记录
- 活动现场图、主视觉图、Logo 或品牌素材

如果图片或 PDF 无法读取，需要补充文字摘要或截图。

## 工作流程

1. 提供内容母版。
2. skill 提炼活动信息和封面钩子。
3. 选择企划风格，或选择“交给模型决定”。
4. 选择信息结构，或选择“交给模型决定”。
5. 选择素材使用方式。
6. 从 3 组封面文案中选择一组。
7. skill 输出设计构思，并提示：

   ```text
   请确认视觉元素。如确认，请告诉我继续。如不确认，请提出你的修改建议。
   ```

8. 你回复“继续”后，skill 生成最终提示词或开始出图。

## 使用方式

在 Codex 中触发：

```text
[$team-activity-xhs-cover] 请帮我做一个团队活动小红书封面
```

然后粘贴活动内容母版，或上传活动资料。

也可以一次给出更多信息：

```text
[$team-activity-xhs-cover]
请基于下面的活动方案生成小红书封面提示词。
风格交给模型决定，信息结构也交给模型决定。

活动内容：
……
```

## 示例输入

```text
我们要组织一场上海城市探索活动，参与者是 30 位新员工。
活动包括分组领取任务卡、寻找城市地标、完成合影挑战、收集印章，
最后在咖啡馆集合分享路线故事并颁奖。
```

## 示例输出节奏

skill 不会一上来直接给最终提示词，而是先提炼：

```text
我先从内容母版里提炼到：
- 活动一句话：
- 适合人群：
- 情绪钩子：
- 主题包装：
- 可视化元素：
- 可用标签：
```

然后逐步让你选择风格、信息结构、素材使用方式和封面文案。确认视觉元素后，才进入最终提示词或出图。

## 安装

```bash

```

目录至少包含：

```text
team-activity-xhs-cover/
├── SKILL.md
└── agents/
    └── openai.yaml
```

## 设计原则

- 一次只问一个问题，减少用户决策负担。
- 封面文案必须来自内容母版提炼或用户确认。
- 主标题必须短、有画面、有企划感。
- 避免普通团建海报、企业 PPT 模板、行政通知口吻。
- 所有最终提示词必须明确写入“3:4 竖版小红书封面”。
