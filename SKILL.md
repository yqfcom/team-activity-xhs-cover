# 团队活动小红书封面 Skill

读取用户提供的内容母版 -> 自动提炼活动钩子和封面文案 -> 逐步让用户做选择题 -> 生成可直接用于图片生成模型的 3:4 小红书封面提示词。

核心目标：把团队活动包装成一个精致、有主题、有审美、有趣的小型活动企划，而不是普通团建、企业海报或廉价模板。

**重要：一次只问一个问题，等用户回答后再问下一个。**

如果用户已经在当前消息提供了某项信息，直接记录并跳过对应问题。最终只套用用户选择的风格模板；不要输出通用模板。

除用户粘贴活动内容、上传内容母版或上传素材外，所有封面决策问题都必须做成编号选择题。不要用开放式问题收集风格、素材、背景、字体、颜色或标题偏好。

---

## 问题流程（逐一问，不要一次全问）

**第1问：内容母版**

如果用户还没有提供内容母版，只问：

请提供内容母版。可以是活动方案文字、宣传文案、PDF、图片、海报、行程表、聊天记录、报名页截图或任意活动资料。我会先从里面提炼封面主标题、副标题、标签和情绪钩子。

收到内容母版后，先输出一段简短提炼，不要立刻生成最终提示词：

```
我先从内容母版里提炼到：
- 活动一句话：
- 适合人群：
- 情绪钩子：
- 主题包装：
- 可视化元素：
- 可用标签：
```

提炼规则：
- 活动一句话：说明这到底是什么活动，不要照抄原文。
- 适合人群：团队成员、客户、社群、亲子、青年人、管理层、新员工等。
- 情绪钩子：松弛、探索、社交、挑战、治愈、仪式感、出片、反差、协作、沉浸、未知感。
- 主题包装：把活动包装成一个故事，如"城市寻宝"、"春日攻略局"、"一日旅行社"、"逃离办公室计划"、"午后灵感采集"、"假日接力计划"。
- 可视化元素：地点、路线、道具、食物、服装、任务卡、地图、贴纸、票根、拍立得、手写字、社交评论卡片。
- 可用标签：每个 2-6 个字，适合放在封面角标、路线节点或贴纸上。

如果用户提供的是图片或 PDF，需要基于可见内容提炼；如果无法读取，要求用户补充文字摘要或截图。

---

**第2问：选企划风格**

列出以下 9 种风格，让用户选一种：

1. **蓝天旅行团风** — 大合照或活动现场做主视觉，蓝色边框、时间地点、横幅副标题，适合轻松出游和多人活动
2. **手账拼贴风** — 多张照片、纸张、地图、票根、贴纸和手写标注，适合城市探索、文化体验、路线活动
3. **沉浸体验风** — 一张强场景照片做底，大手写标题压上去，适合剧场、展览、夜游、非遗、运动挑战
4. **拍立得主题风** — 中心拍立得主图，四周分布手写短句和竖排标签，适合纪念感、人物出镜、主题写真
5. **地图路线风** — 地图、路线节点、头像或任务标签，适合打卡任务、城市漫游、分组闯关
6. **接力计划风** — 大背景地标 + 多个社交截图卡片，适合传播挑战、多人共创、用户接力
7. **攻略鉴真风** — 强现场画面 + 超大手写标题 + 玩法说明，适合攻略、挑战、投票、现场体验
8. **插画拼贴风** — 照片 + 手绘人物/贴纸/符号，适合轻松生活方式、骑行、散步、宠物、春日活动
9. **交给模型决定** — 由模型基于活动内容、素材类型、传播目标和封面文案，统一判断最适合的企划风格

如果用户选择 **交给模型决定**，不要再追问风格；在后续设计构思中明确写出你选择的风格和选择理由。判断时优先考虑：
- 活动是否需要强现场冲击力、文化仪式感或竞技感。
- 用户是否提供主视觉图、多图素材或品牌素材。
- 封面文案更适合招募、纪念、攻略、路线、社交传播还是挑战感。
- 哪个模板能让主标题、素材和情绪钩子形成最统一的画面。

---

**第3问：信息结构**

给用户以下选项：

1. **大标题 + 副标题 + 时间地点** — 适合招募、报名、正式活动发布
2. **大标题 + 3 个活动亮点** — 适合突出体验感和参与理由
3. **大标题 + 路线/任务节点** — 适合城市探索、闯关、打卡
4. **大标题 + 参与者身份标签** — 适合社交、分组、角色扮演、团队协作
5. **大标题 + 社交截图/评论卡片** — 适合传播感、接力感、用户证言
6. **交给模型决定**

---

**第4问：素材使用方式**

问用户是否有额外素材，只给以下选项：

1. **有主视觉图** — 用作封面最大照片或背景
2. **有多张活动素材** — 做拼贴、拍立得、截图卡片或路线节点
3. **有品牌/Logo/主办方素材** — 做角标、联合署名或小徽章
4. **只有内容母版** — 模型根据主题生成场景、贴纸和活动道具
5. **不使用原素材** — 只提炼主题，重新生成完整封面画面

用户选择 1-3 时，只要求按顺序上传图 1、图 2、图 3 等，不额外追问开放描述。若必须说明素材用途，给选择题：

1. **主视觉**
2. **拼贴小图**
3. **品牌角标**
4. **活动道具**
5. **路线节点**

---

**第5问：封面文案确认**

基于内容母版提炼 3 组候选，让用户只选编号：

1. **主标题 A** / 副标题 A / 标签 A
2. **主标题 B** / 副标题 B / 标签 B
3. **主标题 C** / 副标题 C / 标签 C
4. **重新生成 3 组**

标题规则：
- 主标题优先 4-12 个字，短、有画面、有企划感。
- 副标题优先 8-20 个字，说清活动体验、情绪或玩法。
- 标签 3-6 个，每个 2-6 个字。
- 避免"团建"、"活动通知"、"员工福利"这类普通企业话术，除非用户明确要求。
- 优先把活动包装成一个有主题的故事，如"春日攻略局"、"一日旅行社"、"城市寻宝计划"、"松弛感补给站"。

---

**第6问：视觉元素确认**

用户选择封面文案后，先输出一段简短设计构思，不要立刻生成最终提示词，也不要立刻调用图片生成工具。

设计构思必须包含：
- 选定企划风格：如果用户前面选择了"交给模型决定"，写明最终选择的风格和 1 句理由。
- 主视觉：说明使用参考图、生成图或内容母版中的哪类元素。
- 版式结构：说明主标题、副标题、标签、时间地点或角标的大致位置。
- 色彩与质感：说明主色、辅助色、纸张/照片/地图/舞台/贴纸等质感。
- 关键视觉元素：列出 4-8 个会出现在画面里的元素。
- 避免事项：说明最需要避免的 1-3 个问题。

设计构思后，必须追加这句跟踪话术：

请确认视觉元素。如确认，请告诉我继续。如不确认，请提出你的修改建议。

只有当用户回复"继续"、"确认继续"、"可以出图"等明确确认后，才进入最终提示词生成或图片生成。若用户提出修改建议，先更新设计构思并再次要求确认。

## 通用视觉规范

所有风格都遵守：

- 画面比例必须是 **3:4 竖版小红书封面**。
- 视觉第一印象必须是"活动企划/主题故事"，不是公司 PPT、普通团建海报或景区宣传单。
- 主标题是第一视觉优先级，必须清晰、短促、有情绪。
- 信息层级清楚：主标题最大，副标题其次，时间地点、标签、路线节点更小。
- 可以使用真实照片、地图、纸张纹理、拍立得、票根、胶带、便签、涂鸦、贴纸、路线线条、评论卡片。
- 字体可以是手写大字、书法感标题、复古像素字、粗圆体、杂志标题字；不要全用细字体。
- 色彩应有明确主题：天空蓝、旅行粉、奶油黄、复古橙、地图蓝、节日红、草绿、纸张米白都可以，但不要做成单调企业蓝。
- 可加入小红书式红色胶囊标签、活动品牌角标、日期、地点、路线名，但不要让角标抢主标题。
- 所有重要文字、人物脸部、路线节点、标签都要留在安全区内。

避免：

- 不要普通团建海报感、企业年会背景板感、PPT 模板感。
- 不要极简空白到像地产广告。
- 不要过度商务、过度高冷、过度科技。
- 不要让标题遮挡人物眼睛和嘴巴。
- 不要堆太多无法阅读的小字。
- 不要把活动写成"员工福利活动通知"这类行政口吻。

## 生成提示词的核心要求

拿到所有答案且用户确认视觉元素后，按对应风格模板生成提示词；如果用户明确要求直接出图，则基于确认后的设计构思调用图片生成工具。提示词必须做到：

- **内容来自母版**：主标题、副标题、标签、时间地点、活动道具必须来自内容母版提炼或用户确认。
- **标题原样出现**：不要让模型改写用户确认过的标题。
- **空间关系清晰**：前景/中景/背景各是什么，哪个遮挡哪个。
- **素材指代明确**：直接写"参考图1/图2/图3"，不要复述参考图全部细节。
- **企划感明确**：写清它像活动手册、旅行企划、路线任务卡、手账拼贴或社交接力海报。
- **比例写死**：每个最终提示词都必须明确写"3:4 竖版小红书封面"。

## 风格一：蓝天旅行团风

适合轻松出游、团队旅行、户外团体活动、多人合照。

```
使用参考图中的活动人物或活动现场作为主视觉；如果没有参考图，生成一张自然、真实、年轻、有松弛感的团队活动现场照。

画面中央放置一张大幅活动照片，占画面约 65%，照片有白色细边框，像被贴在蓝色活动手册上。人物表情自然、有互动感，整体像一次有主题的团队出行。

画面顶部放置超大中文主标题"[主标题]"，字体为粗手写黑体或综艺感粗字，黑色填充，白色厚描边，略微倾斜，形成强点击感。

主标题下方或照片左上角放置副标题横幅"[副标题]"，横幅为天空蓝或白色，文字清晰，像旅行社活动标语。

左侧或右上角放置时间地点信息：[时间地点]；底部放置 3 个小标签：[标签1]、[标签2]、[标签3]，做成贴纸或胶囊角标。

背景为明亮天空蓝纸张纹理，四角有取景框线条，少量白色涂鸦、箭头和胶带装饰。

3:4 竖版小红书封面，清爽、热闹、真实、有活动企划感，文字清晰可读，重要元素不超出安全区。
```

## 风格二：手账拼贴风

适合城市探索、文化体验、美食路线、展览路线、非遗活动。

```
将活动包装成一本精致的城市探索手账。使用参考图作为拼贴照片；如果没有参考图，生成与活动主题相关的 4-6 张小照片素材。

画面由纸张纹理、地图碎片、票根、胶带、便签、贴纸、小照片组成。照片大小错落、轻微旋转，有真实手工拼贴感。

画面上方或右侧放置主标题"[主标题]"，字体为细长手写标题字或复古杂志标题字，颜色使用亮粉、天蓝、黑色或奶油黄中的 1-2 种，标题占画面约 25%-35%。

副标题"[副标题]"做成小纸条、便签或票根，贴在主照片旁边。时间地点做成印章、胶囊标签或手写角标。

用路线线条串联 3-5 个活动节点：[标签1]、[标签2]、[标签3]、[标签4]，节点像地图打卡点。

加入少量与活动相关的道具贴纸，如相机、饮料、运动装备、花、餐具、任务卡、印章或城市地标。

3:4 竖版小红书封面，像精致活动 moodboard，信息密度高但层级清楚，避免廉价模板感。
```

## 风格三：沉浸体验风

适合夜游、剧场、运动挑战、展览、主题空间、强现场画面。

```
使用参考图1作为整张封面的沉浸式背景；如果没有参考图，生成一个与活动主题强相关的现场场景，画面有动感和参与感。

背景照片铺满画面，局部轻微动态模糊或光影增强，让现场氛围强烈但不影响标题阅读。

画面上方放置超大中文主标题"[主标题]"，字体为粗手写字或书法感标题字，白色或主题高亮色填充，带轻微阴影或描边，占画面约 30%-40%。

副标题"[副标题]"以手写斜排方式放在人物或场景旁边，像现场旁白。下方放置 3 行短句标签：[标签1]、[标签2]、[标签3]，每行不超过 8 个字。

加入少量手绘符号，如放大镜、感叹号、光线、箭头、对话气泡，强调"正在参与"而不是"看广告"。

3:4 竖版小红书封面，沉浸、热烈、有故事，像一个值得加入的现场企划，文字不遮挡关键人物或动作。
```

## 风格四：拍立得主题风

适合纪念感、人物出镜、主题写真、节日活动、复古体验。

```
画面中心放置一张大拍立得照片，使用参考图1作为拍立得内容；如果没有参考图，生成与活动主题相关的人物或场景照片。

拍立得略微倾斜，占画面约 55%，有真实纸张厚度、投影和白色相纸边。照片下方可以写主办方或活动昵称。

背景为单色或轻微纸张纹理，颜色根据活动情绪选择，如旅行粉、奶油黄、浅蓝、复古绿或暖橙。

画面顶部和底部用大号手写中文标题组合成"[主标题]"，黑白混排或主题色混排，像随手写在封面上的活动宣言。

副标题"[副标题]"拆成 2-4 段竖排短句，分布在拍立得四周。标签 [标签1]、[标签2]、[标签3] 做成小贴纸、印章或手写批注。

3:4 竖版小红书封面，复古、有纪念感、有主题写真气质，像一本活动相册的封面，不要像证件照或影楼模板。
```

## 风格五：地图路线风

适合城市漫游、分组闯关、路线打卡、探索任务。

```
将活动包装成一张任务地图或路线企划图。

画面中部放置一张简化地图或插画地图，占画面约 50%，地图上有路线线条、打卡点、任务节点和小图标。节点名称使用：[标签1]、[标签2]、[标签3]、[标签4]。

画面上方放置主标题"[主标题]"，字体为复古像素字、手写标题字或粗圆体，颜色与地图主题匹配，标题下方加一条细线或手绘下划线。

副标题"[副标题]"放在主标题下方或地图下方，像活动说明。时间地点做成车票、指南针、定位标签或小卡片。

如果有参与者、角色或分组信息，把它们做成圆形头像标签、队伍旗帜或地图图钉，分布在路线附近。

右下角可以加入一个手机界面、任务卡或活动通行证小插图，强化参与感。

3:4 竖版小红书封面，像一张可爱的城市探索任务地图，路线清楚，节点可读，有企划感和游戏感。
```

副标题"[副标题]"放在卡片下方或页面底部，像活动发起语。时间地点放在右下角小角标。

## 风格七：攻略鉴真风

## 通用写提示词规范

1. 先写画面，再写风格：主体、标题、素材、背景、装饰、构图顺序要清楚。
2. 主标题、副标题、标签必须原样出现。
3. 参考图直接编号：写"参考图1/图2/图3"，不要长篇复述。
4. 短字优先：封面文字必须适合一眼读完。
5. 保持活动企划感：像旅行手册、城市任务卡、主题活动 moodboard 或社交接力海报。
6. 最终只输出一个完整提示词，除非用户要求多个版本。
